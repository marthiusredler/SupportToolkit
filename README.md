# Support Toolkit

> Ferramenta desktop para Windows 10/11 desenvolvida para auxiliar analistas de Suporte Técnico em diagnósticos, coleta de informações do sistema, gerenciamento de certificados digitais e execução de tarefas recorrentes.

---

# Sobre o Projeto

O **Support Toolkit** nasceu da necessidade de centralizar em uma única aplicação diversas ferramentas utilizadas diariamente por equipes de suporte técnico, especialmente em ambientes que trabalham com **Certificados Digitais ICP-Brasil**.

O objetivo é reduzir o tempo gasto na coleta de informações, facilitar diagnósticos, automatizar procedimentos repetitivos e padronizar o atendimento.

A aplicação será desenvolvida em **Python** utilizando **PySide6** para interface gráfica, com foco em Windows 10 e Windows 11.

---

# Objetivos

- Centralizar ferramentas de suporte técnico.
- Facilitar a parametrização de computadores.
- Auxiliar na identificação de problemas.
- Gerar relatórios técnicos.
- Automatizar tarefas repetitivas.
- Diminuir o tempo de atendimento.
- Melhorar a produtividade da equipe de suporte.
